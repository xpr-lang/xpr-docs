
## API

### `new Xpr()`

Creates a new XPR engine instance.

### `engine.evaluate(expression, context?)`

Evaluates an XPR expression against an optional context object.

### `engine.addFunction(name, fn)`

Registers a custom function for use in expressions.

See [xpr-lang/xpr-js](https://github.com/xpr-lang/xpr-js) for the full implementation.
